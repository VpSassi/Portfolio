Olen vuonna 1994 syntynyt ja alunperin Järvenpäästä kotoisin, mutta asun nykyisin Mikkelissä. Harrastuksina minulla on lenkkeily ja videopelien pelailu. Valmistuin Kaakkois-Suomen Ammattikorkeakoulusta joulukuussa 2020 ja olen sitä kautta tutustunut erilaisiin ohjelmointitekniikoihin ja kieliin. Olen opintojeni aikana tutustunut muun muassa Node.js:ään, Ioniciin, Angulariin ja taidan tyypillisesti HTML-merkkikielen ja CSS:än. Suurin osaamiseni on front-end ohjelmointi JavaScriptin/TypeScriptin avulla, yhdistettynä React-komponenttikirjaston työkaluihin. Jonkin verran kokemusta minulla löytyy myös C# -ohjelmointikielestä Unity peliprojektien osalta.

Työkokemusta minulla on Mindhive Oy:stä, kouluni työharjoittelujaksolta, sekä opinnäytetyöni osalta.
